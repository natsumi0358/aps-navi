# ②アカマネ機能（会社調査支援）

初回商談前の会社調査を支援するシステム。
APS研修で教えている会社調査の必須項目をフォームで入力し、
その情報を①提案アドバイザーAIに渡してアドバイスを生成する。

参考：https://www.akamane.jp/

---

## これから作るもの

### 画面構成（予定）
1. **会社情報入力フォーム** — 会社名・業種・規模・HP URL など
2. **APS必須調査項目チェックリスト** — 以下の項目を入力
   - 会社概要（事業内容・売上規模・従業員数）
   - 中期経営計画・方針
   - キーパーソン一覧（役職・意思決定権の有無）
   - 現在使っているシステム一覧（自社・他社・リプレース時期）
   - 競合状況
   - 顧客の顧客（エンドユーザー）の課題
   - 潜在ニーズ・本音（ヒアリングメモ）
3. **AIアドバイス生成ボタン** — 入力情報を①に渡してアドバイスを表示
4. **調査履歴一覧** — 過去に調査した会社の一覧

### 技術スタック（予定）
- Flask（Python）
- SQLite（調査データの保存）
- Claude API（①と共通）
