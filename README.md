# APSナビ — アカウントプランニングセッション作成サポートツール

N.K.ナーツ株式会社の営業チーム向けAPS作成支援Webアプリ。

## 機能

- **HP自動取得** — 会社HPのURLを入力するとAIがWork6（お客様概要）を自動入力
- **業界分析AI生成** — PEST・5Forces・SWOT・クロスSWOT・ポジショニングを一括生成
- **APS入力フォーム** — Work6〜13に準拠した全フィールドを搭載
- **PPTXダウンロード** — テンプレートに情報を流し込んでPPTX出力
- **AIアドバイス** — 入力情報をもとにAPS観点のアドバイスとチャット
